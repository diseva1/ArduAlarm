# ArduAlarm

ArduAlarm es un proyecto de alarma basado en arduino.

## Componentes

 - Arduino Uno
 - Keypad 4x4
 - Modulo GSM 800L
 - Cargador de baterias LIPO
 - Rele 220v
 - LCD I2C 16x2
 - Sensor para puertas
